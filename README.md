\# AI Gesture Stage v3.0



A real-time gesture recognition and animation system built using Python.



\## 🔥 Features

\- Multi-hand tracking (MediaPipe)

\- Real-time gesture recognition

\- Particle-based animation system

\- Interactive visual effects



\## 🛠️ Technologies Used

\- Python

\- OpenCV

\- MediaPipe Tasks API

\- NumPy



\## ▶️ How to Run

pip install opencv-python mediapipe numpy  

python HandGesture.py
